============================================

move-window.sh
==============

Is a simple script to manage window positions using keyboard shortcuts.
You can set the keyboard shortcuts like:
    left        ->      <Super> + 4
    top-left    ->      <Super> + 7
    bottom-left ->      <Super> + 1
    right       ->      <Super> + 6
    top-right   ->      <Super> + 9
    bottom-right->      <Super> + 3
    center      ->      <Super> + 5
